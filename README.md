**ChillGold** est le sibling de la galaxie [InvestChill](https://github.com/investchill)
dédié à la classe **gold** (or physique via ETC UCITS). Il s'inscrit dans le folio
All-Weather [ChillFOLIO](https://github.com/investchill/chillfolio) en couvrant le quadrant
macro **croissance basse / inflation haute** — la stagflation, seul quadrant non couvert
par les actions ni les treasuries longues.

> ⚠️ **Phase amorçage.** Le projet est tout neuf. Le signal mensuel mécanique, le choix
> d'ETC concret et le backtest sibling-spécifique restent à figer.

## Cadre

- **Quadrant macro adressé** : croissance basse, inflation haute (stagflation).
  Seul actif couvrant ce quadrant dans le folio cible ChillFOLIO.
- **Cadre fiscal France** : PEA non-éligible — CTO uniquement (PFU 31,4 % en 2026
  sur les plus-values lors de la cession de l'ETC). L'or physique particulier
  (lingots, pièces) relève d'un régime spécifique et n'est pas dans le périmètre
  de ce sibling.
- **Mécanique attendue** : signal mensuel ON/OFF, anti-overfit (1-2 paramètres
  maximum, plateau de stabilité, walk-forward), gestion explicite des régimes
  long-baissiers historiques (drag séculaire post-Volcker 1980-1999).
